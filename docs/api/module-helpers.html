inBound

(inner) inBound(n, maxopt, minopt) → {boolean}

Checks if a number is within a bound

Parameters:
Name Type Attributes Default Description
n number
max number optional
 1
min number optional
 0
Returns:

returns true if the number is smaller than max and bigger than min

Type
boolean
Example
inBound(5) // => false
inBound(-5) // => false
inBound(1) // => false
inBound(0) // => false
inBound(0.1) // => true
inBound(0.9999) // => true
inBound(4.9, 5) // => true
inBound(-4.9, -5) // => true
inBound(10, 10) // => false
