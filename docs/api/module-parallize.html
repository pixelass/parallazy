parallize

(inner) parallize(element, options) → {object}

Checks visibility of element in viewport and returns offset as numbers.

Parameters:
Name Type Description
element HTMLElement
options object
Properties
Name Type Description
entering boolean

track while entering and leaving
offsetX number

bounding box-x on viewPort
offsetY number

bounding box-y on viewPort
decimals number

number of decimals to return
Returns:

{top: [0...1], left: [0...1], right: [0...1], bottom: [0...1]}

Type
object
Example
const el = document.querySelector('.test')
parallize(el)
parallize(el, {
  entering: false
})
parallize(el, {
  offsetX: 10,
  offsetY: 100
})
parallize(el, {
  decimals: 10
})
